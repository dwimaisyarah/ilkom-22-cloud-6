Belajar Tag-Tag HTML untuk Pemula

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web.

Contoh Struktur HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini paragraf pertama.</p>
  </body>
</html>

Tag-Tag Dasar

  • <h1> - <h6> : Judul (Heading)
  • <p> : Paragraf
  • <a> : Tautan (Link)
  • <img> : Gambar
  • <ul>, <ol>, <li> : Daftar

Contoh Tautan dan Gambar

<a href="https://www.google.com">Kunjungi Google</a>

<img src="gambar.jpg" alt="Contoh Gambar" width="200">

Penutup

HTML adalah dasar dari setiap halaman web. Dengan memahami struktur dan tag-nya, kamu sudah memulai langkah awal menjadi web developer.
