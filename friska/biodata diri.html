Apa Itu Cloud Computing?

Pemanfaatan Teknologi Komputasi Awan

Definisi

Cloud computing (komputasi awan) adalah teknologi yang memungkinkan pengguna mengakses sumber daya komputasi (seperti server, penyimpanan, database, dan perangkat lunak) melalui internet.

Model Layanan

  • IaaS (Infrastructure as a Service) – Contoh: AWS EC2, Google Compute Engine
  • PaaS (Platform as a Service) – Contoh: Google App Engine, Heroku
  • SaaS (Software as a Service) – Contoh: Google Docs, Dropbox

Manfaat Cloud Computing

  • Efisiensi biaya
  • Skalabilitas
  • Akses data dari mana saja
  • Backup dan pemulihan data

Contoh Penggunaan

Mahasiswa menggunakan Google Drive untuk menyimpan tugas, perusahaan mengelola server dengan AWS, dan tim pengembang membuat aplikasi dengan Firebase.
